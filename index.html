LUCIS Land Use Conflict Identification Strategy

A spatial planning tool for land use conflict analysis

Setup Instructions

Step 1: On your local computer, create a folder called LUCIS in your desired location.

Step 2: Within the LUCIS folder, create the following subfolders:

  • All Data - for input shapefiles and Excel data
  • Ranked Tifs - for processed raster outputs
  • [Study Area Name] - replace with your study area name (e.g., Maun, Seronga, KAZA)

Step 3: Place your data files in the appropriate folders:

  • Excel data file (Data.xlsx) in the All Data folder
  • Shapefiles and other geospatial data in subfolders within All Data
  • Study area boundary shapefile in the study area folder

Step 4: Click the button below to select your LUCIS base directory.
